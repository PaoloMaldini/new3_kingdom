# 《楼台烟雨》操作手册

**版本：** 1.0
**日期：** 2026年4月

---

## 一、游戏简介

《楼台烟雨》是一款以中华古代建筑为主题的水墨风互动体验游戏。玩家通过水墨风格的交互式地图，探索中国古代建筑，点击建筑标记进入历史场景，体验建筑背后的故事。

### 1.1 游戏入口

打开游戏后，在首页可以看到两个入口按钮：
- **⚔️ 三国地图**：体验三国时期的战略剧本
- **🏛️ 古建筑地图**：探索中国古代建筑（本游戏主要模式）

---

## 二、首页操作

### 2.1 首页布局

首页采用居中对齐的卡片式设计，中间展示游戏标题和入口按钮。

### 2.2 进入游戏

**操作步骤：**
1. 在首页点击 **「🏛️ 古建筑地图」** 按钮
2. 进入水墨风古建筑地图页面

---

## 三、古建筑地图操作

### 3.1 地图界面

古建筑地图页面包含以下元素：

```
┌─────────────────────────────────────────────────┐
│ [← 返回]     古建筑地图         [分类筛选按钮]    │  ← 头部区域
├─────────────────────────────────────────────────┤
│                                                 │
│    ○  ○        ○                    ○          │  ← 建筑标记
│        ○              ○                         │
│                          [右侧:建筑列表]        │  ← 侧边栏
│    ○        ○            ○       [建筑名称]    │
│                                                 │
├─────────────────────────────────────────────────┤
│  图例: 🏠民居  🏛️官府  ⛩️皇宫  🌉桥梁         │  ← 图例
└─────────────────────────────────────────────────┘
```

### 3.2 分类筛选

**操作方法：**
- 点击页面顶部的分类按钮（民居/官府/皇宫/桥梁）
- 选中的分类会以金色高亮显示
- 点击「全部」可显示所有建筑

**分类图标对照：**
| 图标 | 分类 | 示例建筑 |
|-----|------|---------|
| 🏠 | 民居 | 福建土楼、北京四合院 |
| 🏛️ | 官府 | 天一阁、包拯祠、黄鹤楼 |
| ⛩️ | 皇宫 | 故宫、大明宫、阿房宫 |
| 🌉 | 桥梁 | 赵州桥、断桥、卢沟桥 |

### 3.3 查看建筑详情

**方法一：点击地图标记**
1. 将鼠标移至地图上的建筑标记（○）上方
2. 标记会放大并显示墨点涟漪动画
3. 下方会弹出提示框，显示建筑名称、时代、位置、简介
4. 点击提示框中的「查看详情」进入详情页

**方法二：点击侧边栏**
1. 查看页面右侧的「建筑一览」列表
2. 点击想要查看的建筑条目
3. 直接进入该建筑的详情页面

### 3.4 地图交互

| 交互动作 | 桌面端 | 移动端 |
|---------|-------|-------|
| 悬停查看 | 鼠标悬停显示提示 | 点击标记显示提示 |
| 缩放地图 | 滚轮缩放（预留） | 双指缩放（预留） |
| 平移地图 | 拖拽移动 | 滑动移动 |

建筑详情页提供三个信息卡片：

1. **建筑简介**：建筑的基本描述和外观特点
2. **历史意义**：建筑的历史背景和文化价值
3. **关联人物**：与该建筑相关的历史人物标签

**操作步骤：**
1. 在建筑详情页阅读「剧本简介」
2. 点击「开始体验」按钮（位于信息卡片下方）
3. 进入剧本互动页面

## 五、剧本互动操作

### 5.1 剧本页面元素

剧本卡片包含以下元素：
- **角色头像**：圆形显示主角首字
- **事件标题**：当前事件的名称
- **年份标签**：故事发生的年代
- **场景描述**：200字左右的场景叙述
- **选项按钮**：左右两个选择按钮
- **属性条**：四项属性的当前数值

### 5.2 桌面端操作

**鼠标点击选择：**
1. 阅读场景描述和选项
2. 将鼠标移至想要选择的选项上
3. 选项会高亮显示
4. 点击确认选择

### 5.3 移动端操作

**滑动选择：**
1. 用手指按住屏幕上的剧本卡片
2. 向左滑动查看左侧选项详情
3. 向右滑动查看右侧选项详情
4. 滑动超过阈值（80px）松手，确认选择
5. 也可以直接点击左右两侧进行选择

**滑动阈值：**
| 滑动距离 | 结果 |
|---------|-----|
| < 30px | 忽略滑动 |
| 30-80px | 显示预览 |
| ≥ 80px | 确认选择 |

### 5.4 属性变化反馈

做出选择后，系统会显示属性变化：

| 变化类型 | 视觉效果 |
|---------|---------|
| 属性增加 | 绿色高亮 + "+N" |
| 属性减少 | 红色高亮 + "-N" |
| 达到极值 | 红色警告提示 |

### 5.5 继续游戏

选择后，系统会：
1. 显示属性变化动画（约1.2秒）
2. 自动进入下一个剧本事件
3. 或者进入结局画面

## 六、结局与重玩

### 6.1 结局触发

剧本共有2个事件，完成后显示结局。

**触发结局的条件：**
- 完成所有剧本事件
- 任意属性达到100（极致结局）
- 任意属性降至0（耗尽结局）

### 6.2 结局画面

结局画面显示：
- 剧本体验结束提示
- 四项属性的最终数值
- 「再体验一次」按钮（重玩当前建筑）
- 「返回地图」按钮（返回地图选择其他建筑）

### 6.3 重玩操作

**重新体验当前建筑：**
1. 在结局画面点击「再体验一次」
2. 属性重置为初始值（各50）
3. 从第一个事件重新开始

**体验其他建筑：**
1. 点击「返回地图」
2. 返回古建筑地图
3. 选择其他建筑

## 七、三国地图操作

### 7.1 进入三国地图

在首页点击「⚔️ 三国地图」按钮进入。

### 7.2 选择主公

三国地图提供5位主公供选择：
- 曹操（乱世奸雄）
- 刘备（仁义双剑）
- 孙坚（江东之虎）
- 袁绍（联军盟主）
- 董卓（苦命黄莲）

**操作：** 左右滑动选择主公卡片，点击「选择此主公」确认。

### 7.3 选择地点

进入地图后，点击地图上的位置标记或侧边栏列表，选择要体验的地点剧本。

### 7.4 剧本流程

三国地图的剧本流程与古建筑地图类似，但：
- 属性系统为：天意、权谋、兵力、粮草
- 剧本更长，事件更多
- 有随机侧事件的分支机制

## 八、界面响应式说明

### 8.1 桌面端（≥900px）

- 双栏布局：地图+侧边栏
- 鼠标悬停显示提示
- 点击选择确认

### 8.2 平板端（640-899px）

- 单栏布局：地图全宽
- 侧边栏隐藏（可展开）
- 触摸交互为主

### 8.3 手机端（<640px）

- 紧凑布局
- 建筑列表改为底部抽屉
- 全屏卡片显示剧本
- 滑动操作为主

## 九、常见问题

### Q1: 为什么滑动没有反应？

**可能原因：**
- 滑动距离太短（需要超过80px）
- 手机开启了系统手势（尝试关闭系统手势）
- 网络延迟（刷新页面重试）

**解决方法：**
1. 尝试使用点击选择代替滑动
2. 刷新页面重试
3. 检查网络连接

### Q2: 图片加载失败怎么办？

**可能原因：**
- 网络图片CDN访问问题
- 浏览器缓存问题

**解决方法：**
1. 刷新页面
2. 清除浏览器缓存
3. 使用其他浏览器尝试

### Q3: 如何分享我的游戏成就？

**当前版本：** 分享功能正在开发中

**暂时方案：**
1. 截图游戏画面
2. 手动分享到社交平台

### Q4: 游戏支持离线玩吗？

**当前版本：** 需要网络连接加载资源

**未来计划：** 支持PWA离线模式
